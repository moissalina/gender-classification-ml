# gender-classification-ml
Проект по бинарной классификации пола человека на основе числовых признаков внешности.

Задача: предсказать пол человека (`gender`) на основе параметров лица
(волосы, лоб, нос, губы и расстояния между элементами).

Проект охватывает полный цикл анализа данных:
- предобработка данных
- исследовательский анализ (EDA)
- обучение моделей машинного обучения
- оценка качества моделей

## Используемые данные
Датасет содержит **5001 наблюдение** и **8 признаков**:

**Числовые признаки:**
- `long_hair`
- `forehead_width_cm`
- `forehead_height_cm`
- `nose_wide`
- `nose_long`
- `lips_thin`
- `distance_nose_to_lip_long`

**Целевая переменная:**
- `gender` — категориальный признак (Male / Female)
